# OpenBird Examples

Usage examples for [OpenBird](https://www.npmjs.com/package/openbird) — Feishu infrastructure service (WebSocket event forwarding + MCP Server).

## Prerequisites

- Node.js >= 20
- A Feishu account cookie (see individual examples for details)

## Examples

| Example | Description |
|---|---|
| [basic-usage](./basic-usage/) | Receive real-time Feishu messages via webhook + call Feishu API tools via MCP |

## How to use

Each example is a standalone project. `cd` into the example directory and follow its README.
